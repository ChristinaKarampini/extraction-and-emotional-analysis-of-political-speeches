# extraction-and-emotional-analysis-of-political-speeches

Αυτό το project ασχολείται με την εξαγωγή και ανάλυση συναισθημάτων σε πολιτικές ομιλίες του Ηνωμένου Βασιλείου.

Βήματα του έργου

1.Δημιουργία CSV (Extract data in CSV file.py)
Από τον φάκελο με τα πρακτικά του Κοινοβουλίου του Ηνωμένου Βασιλείου, γίνεται εξαγωγή δεδομένων από τα αρχεία XML.
Τα δεδομένα συσχετίζονται και οργανώνονται σε λίστες ώστε να εξαχθούν σωστά σε ένα αρχείο CSV με όλες τις σημαντικές πληροφορίες.

2.Δημιουργία δείγματος ομιλιών (Text 300 sample)
Από το συνολικό αρχείο που περιέχει όλες τις ομιλίες ανά έτος, δημιουργείται ένα τυχαίο δείγμα 300 ομιλιών για περαιτέρω ανάλυση.

3.Εκπαίδευση του μοντέλου (Train the model)
Το μοντέλο εκπαιδεύεται με few shot learning για να αναγνωρίζει συναισθήματα και συναισθηματικές τάσεις.

4.Εφαρμογή του μοντέλου (Model inference)
Το εκπαιδευμένο μοντέλο εφαρμόζεται στο δείγμα των 300 τυχαίων ομιλιών για να εξαχθούν αποτελέσματα ανάλυσης συναισθήματος.
